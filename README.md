# Student Performance Prediction Using ML

Predicting student academic performance using Multiple Linear Regression.

## Dataset
- Source: Kaggle (Nikhil7280)
- 10,000 student records | 6 features

## Features Used
- Hours Studied
- Previous Scores
- Extracurricular Activities
- Sleep Hours
- Sample Question Papers Practiced

## Model Results
- ✅ R² Score: 98.90%
- ✅ RMSE: 2.02
- ✅ MSE: 4.08
- 🏆 Top Predictor: Hours Studied

## Tools Used
Python | Pandas | Scikit-learn | Matplotlib | Seaborn | Kaggle Notebooks
